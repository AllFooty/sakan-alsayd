# Sakan Alsayd - Student & Professional Housing Platform

## Overview
A bilingual (English/Arabic) Next.js website for Sakan Alsayd, a student and professional housing company in Saudi Arabia. The site showcases fully furnished and serviced housing for female students and working women.

## Tech Stack
- **Framework**: Next.js 16.1.6 with Turbopack
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Internationalization**: next-intl (Arabic & English)
- **Animations**: Framer Motion
- **Forms**: React Hook Form with Zod validation
- **Icons**: Lucide React

## Project Structure
```
src/
├── app/              # Next.js App Router pages
├── components/       # React components
├── data/             # Static data (contacts, locations, services, testimonials)
├── i18n/             # Internationalization configuration
├── lib/              # Utility functions
├── messages/         # Translation files (ar.json, en.json)
└── middleware.ts     # Next.js middleware for i18n routing
public/               # Static assets
```

## Development
- **Dev Server**: `npm run dev -- -p 5000 -H 0.0.0.0`
- **Build**: `npm run build`
- **Start**: `npm run start`
- **Port**: 5000
